# type-challenges-judge

[type-challenges](https://github.com/type-challenges/type-challenges)のオンラインジャッジです。

![](https://i.gyazo.com/e9eff32dc0db479da0a31eef62ebdd21.png)
## できること

- type-challengesの問題の閲覧
- 問題の回答の提出、正誤判定
- 自分がどれくらい問題を解いたかをグラフで確認


## 環境構築

```bash
# フロントエンド
yarn install
yarn run dev

# Firebaseのエミュレーターの起動
yarn run emulators
yarn run dbseed
```
